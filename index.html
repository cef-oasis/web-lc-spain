Linked Connections in Spain

Transforming open Spainsh GTFS to Linked Data


Using the Linked Connections specification and open GTFS datasets available in different open data portals in Spain we provide useful Linked Data to route planner developers.

We also provide the metadata of this datasets following TransportDCAT-AP, an application profile of DCAT-AP developed during the CEF-OASIS project (2016-2018)
